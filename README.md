# tradevault
Cloud-native trade journal platform built with Python, Kubernetes, Terraform, AWS, and GitOps.

## Tech Stack
- Python FastAPI
- PostgreSQL
- Docker
- Kubernetes
- Helm
- GitHub Actions
- ArgoCD
- Terraform
- AWS
- Prometheus
- Grafana

## Current Features
- FastAPI backend with CRUD APIs
- PostgreSQL database
- SQLAlchemy ORM
- Dockerized API
- Docker Compose local stack
- Persistent PostgreSQL volume
- Health checks
- GitHub Actions CI
- Docker Hub image publishing
